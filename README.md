# ИИ-агент по автоматизации проверки кода студентов

Критерии вычитываются из README.md, а сам код из блокнота solution.ipynb.

## Основные функции
- code-review кода студентов и простановка баллов при открытии и переоткрытии Pull Request в GitHub;
- проверка активности загрузки решений (не чаще 1 раза в минуту);
- отправка алертов на почту при ошибке работы агента.


## Демонстрация работы агента

Работу агента можно посмотреть в этой ветке pull request тестового репозитория: 
https://github.com/Sofia-Oleynik/test_code_review_agent/pull/2


## Установка

### Предварительные условия
Убедитесь, что у вас установлены Python 3.7 или выше и pip.

### Клонирование репозитория
```bash
git clone https://github.com/Sofia-Oleynik/agent_code_review.git
cd agent_code_review
```


### Установка зависимостей

Установите необходимые зависимости, выполнив:
```bash
pip install -r requirements.txt
```
